## ✨ 核心功能
### 🖥 可视化界面
- 拖拽式交互设计，直观管理菜单层级结构
- 实时预览菜单布局与样式修改（所见即所得）

### 🔄 径向菜单管理
- ​**创建** 多级嵌套菜单

- ​**修改** 菜单标签/排序
- ​**删除** 冗余菜单项

### 🔍 智能命令搜索
- 模糊搜索快速定位Plasticity命令
